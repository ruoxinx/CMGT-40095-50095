Notebook guide

Using the course in Google Colab

Google Colab is the recommended way to use this course. Most notebook-based modules can be opened in the browser with one click, so students usually do not need to install anything.

Recommended workflow

  1. Open the module hub.
  2. Choose the module for the week.
  3. Click the `Launch in Colab` button on that module page.
  4. Sign in with a Google account if prompted.
  5. Run the notebook from top to bottom, one cell at a time.

Tips for beginners

  • Read the explanation text before running the next code cell
  • Run one cell at a time
  • If something breaks, restart from the top instead of skipping ahead
  • Focus on the construction idea first, then the code
