AI in Construction

This course introduces practical AI workflows for construction, architecture, and the built environment through guided notebooks, project examples, and real datasets. The site is organized for students who may be new to coding and want a clear path into the material.

Start here

A clearer path for students with little or no coding background

The course now emphasizes direct access, practical naming, and scaffolded learning. Students can open the main notebooks in Google Colab, run cells in order, and focus on construction ideas rather than setup.

How students can use this site

  • Open a module page and read the learning focus first
  • Click the Colab button to launch the notebook in the browser
  • Run one cell at a time and use markdown explanations as guidance
  • Return to datasets, assignments, and project examples as needed

What makes it more convenient now

  • Module names are written for construction practice, not just computer science
  • Each topic has its own dedicated page with direct links and context
  • Google Colab is treated as the default student workflow
  • Support sections help students know where to begin and who the course serves
