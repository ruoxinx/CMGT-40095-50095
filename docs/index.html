Getting started

Most course notebooks run in Google Colab. Students can open a notebook in the browser and work through it without local installation.

How to get started

  • Begin with the module page for the current topic
  • Review the learning goals and resources
  • Use the Google Colab link to open the notebook in your browser
  • Work through the notebook step by step, then return to assignments, datasets, and project examples as needed

Course tools and resources
